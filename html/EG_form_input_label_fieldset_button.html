• Form表单（表单元素用于获取用户的输入数据）

  • 属性
  • action表示提交的页面（eg：http://www.baidu.com）
  • method表示表单请求方式，有post和get（默认）两种。使用post后台才可以接收到数据
  • name设置表格，方便调用
  • target设置提交时的目标位置：_blank(新窗口) _parent(框架页面中的当前整个窗口) _self(当前窗口) _top(框架页面中的上部窗口)
  • autocomplete设置浏览器记住用户输入的数据，默认on
  • novalidate设置是否执行客户端数据有效性检查

  • input元素

  • autofocus表示光标聚焦
  • disabled表示禁用input元素
  • autocomplete单独设置input的自动完成功能
  • name必须写，定义名称，以便接收相应的值
  • type
  • form表单外的input元素与指定的表单挂钩提交
  • value默认初始值

  • label元素

  • 更好的用户体验，点击标签也可以进行光标聚焦输入
  • for属性，配合input的id标识，在只对标签进行属性设置时也绑定输入体验

  • fieldset分组元素

  • name属性
  • form
  • disabled禁用分组
  • legend设置左上角分组名

  • button

  • type属性submit（默认）,button ,reset
  • submit额外属性(覆盖form属性)：formaction，formmethod,formtarget,formenctype
  • css中设置cursor:pointer，光标移到button上显示手形
